HTML & CSS Study Flashcards

<html>
Root of the HTML document. (Block)
<head>
Contains metadata, styles, and scripts. (Metadata)
<title>
Sets the browser tab title. (Metadata)
<body>
Main content of the web page. (Block)
<link>
Links to external CSS files. (Void)
<h1> - <h4>
Heading levels. (Block)
<ul>
Unordered list. (Block)
<ol>
Ordered list. (Block)
<li>
List item. (Block)
<br>
Line break. (Void)
<p>
Paragraph of text. (Block)
<img>
Displays an image. (Inline)
<a>
Creates a hyperlink. (Inline)
<div>
Generic block container. (Block)
<iframe>
Embeds another webpage. (Inline)
<!-- comment -->
HTML comment. (Ignored)
/* comment */
CSS comment. (Ignored)

Sets an image as background.
background-repeat
Controls if/how background image repeats.
background-position
Sets starting position of background image.
background-color
Sets background color of an element.
What is the current HTML standard?
HTML5. Not all pages follow it. Use a validator like https://validator.w3.org to check.
Why is CSS necessary?
It separates content (HTML) from presentation (CSS), allowing more control and cleaner code.
What does cascading mean in CSS?
It refers to how CSS rules apply in order of specificity and location (inline > internal > external).
How do you link a CSS file?
Using <link rel="stylesheet" href="styles.css"> in the <head>.
Inline vs Block Elements?
Inline doesn't break the flow; block takes full width and starts on a new line.
